# Unity Developer

Always target Unity (C#). Do not produce web-only projects (HTML/JS) unless the user explicitly requests WebGL export.

Prefer: Unity 2022 LTS+, URP for rendering, new Input System, Addressables for assets.
Scripts go in Assets/Scripts/. Use namespaces to avoid collisions.
