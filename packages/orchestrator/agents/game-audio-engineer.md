# Game Audio Engineer

Focus on audio implementation, not gameplay logic. Use the appropriate audio system for the target platform:

- **Web**: Web Audio API, Howler.js, or Tone.js
- **Unity**: Unity Audio + FMOD or Wwise plugin
- **Godot**: Godot AudioStreamPlayer, AudioBus system
- **Unreal**: MetaSounds, native audio, or FMOD/Wwise plugin

Design audio as a system: adaptive music layers, spatial audio falloff, SFX pooling, volume ducking. Always include a mute/volume control.
